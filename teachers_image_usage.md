# دليل استخدام صورة المدرسين الموحدة

تم تطبيق التعديلات لجعل صورة قسم المدرسين (Teachers) اختيارية، مع عرض صورة افتراضية موحدة.

## 1. رفع الصورة الافتراضية

استخدم الـ Endpoint التالي لرفع الصورة التي ستظهر لجميع المدرسين:

**الرابط:**
`POST {{base_url}}/api/admin/system-settings/upload-image`

**الـ Body (form-data):**
| Key | Type | Value | Description |
| --- | --- | --- | --- |
| `key` | Text | `teachers_default_image` | **ضروري** |
| `image` | File | (اختار الصورة) | ملف الصورة |

---

## 2. النتائج المتوقعة

- عند تخزين إعلان مدرس بدون صورة، سيتم قبوله بنجاح.
- واجهة المستخدم ستعرض الصورة الافتراضية في (قائمة الإعلانات - تفاصيل الإعلان - قائمة أفضل المعلنين).
- لتغيير الصورة في المستقبل، فقط أعد رفع صورة جديدة باستخدام نفس الـ endpoint وسيتم التحديث فوراً لدى الجميع.
